About the Project

This project represents an early stage for a simplified Smart Home Control Panel (SHCP). As such, there are some sensors and devices that an SCHP user can trigger or control manually. First off, there is a 'Temperature & Air Humidity' sensor, that gives relative readings, and can potentially trigger a fan if the temperature is too high. Secondly, there is an 'Air Quality' sensor that checks for the dust particles to evaluate the air purity. Once the readings are too high, for the purpose of testing, the sensor triggers a fan as well. Thirdly and lastly, there is a 'Motion' sensor, that is intended for detecting any motion in a closed space, and trigger LED lights on - the LEDs are turned off once no motion has been detected for over a minute.

Why

  • Explores integration of sensor devices into smart home technologies.
  • Unified by a simplified, user-friendly control panel.
  • Prototype demonstrates interaction between sensors and devices using Raspberry Pi and Pico.
  • Aims to create efficient and responsive living environments.

The Process

  • Uses Raspberry Pi as the central server to manage sensor data and serve a web interface.
  • Raspberry Pi communicates with smart devices via WiFi and HTTP protocols.
  • Raspberry Pi Pico handles real-time sensor and LED interactions.
  • Monitors and controls air temperature, humidity, air quality, motion, and devices.

Motivation

  • Targeted at individuals enhancing their living spaces with smart technology.
  • Simplifies household tasks and improves energy efficiency.
  • Provides environmental awareness for better living.
  • Automates lighting, air quality control, and more for convenience and practicality.

Sensors Used

Air Temperature and Humidity

This sensor monitors environmental temperature and humidity levels in real-time. It provides data that could trigger automated actions, such as turning on a fan or adjusting connected devices when temperatures or humidity reach specific thresholds.

Motion Sensor

The motion sensor detects movement within a designated area. When motion is detected, it automatically turns on LED lights for visibility and convenience. After no motion is detected for a set period (e.g., one minute), the lights turn off to conserve energy.

Air Quality Sensor

This sensor evaluates the air for particulate matter to determine air purity levels. If the air quality falls below acceptable thresholds, it can trigger a fan or other devices to improve air circulation, ensuring a healthier environment.

LED Strip

The LED strips provide visual feedback and ambient lighting. They are connected to the motion sensor to turn on when motion is detected. Additionally, users can control their colors and on/off states manually or through automation rules on the server.

Other Technologies Used

Smart Plug (TP-LINK KASA)

The TP-LINK Kasa smart plug is integrated into the system to control connected devices remotely. Users can rename the plug based on its connected device, such as a fan or lamp. Automation rules allow the plug to react to sensor readings, enabling intelligent control of household appliances.

WIFI/HTTP

The system uses WiFi and HTTP protocols for seamless communication between sensors, devices, and the server. This technology ensures data from the sensors is streamed in real-time, enabling users to monitor readings, control devices, and set automation rules through the web interface.
